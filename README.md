# Node Health & Telemetry Server

### Install
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/4UPanElektryk/SimpleWebDash/refs/heads/master/tempserver/deploy.sh)"
```
This script automatically creates a user and downloads the server

don't forget to edit the `/etc/nhts.env`

```env
PORT=3000
```
